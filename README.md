# Python File Automation Script

## Project Description
This project is a Python automation script that performs basic file operations such as creating files, reading files, renaming/moving files, and handling CSV data.

The script demonstrates automation using Python modules like `os`, `shutil`, and `csv` along with proper exception handling.

---

## Features
- Create directories automatically
- Create and write text files
- Create and manage CSV files
- Read file contents
- Rename and move files
- Exception handling for file operations

---

## Technologies Used
- Python
- OS Module
- Shutil Module
- CSV Module

---

## How to Run

1. Clone the repository or download the files.

2. Run the script:

```bash
python main.py
```

---

## Sample Output

The script successfully:
- Creates sample files
- Reads file contents
- Moves and renames files
- Reads CSV data
